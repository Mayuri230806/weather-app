Weather App (Django + REST API)

A weather application built using Django and basic REST APIs that fetches real-time weather data for any city.
The backend processes external API responses and exposes them through RESTful endpoints.
The frontend displays temperature, weather conditions, and location details in a clean interface.

*Features

City-based weather search

Real-time API data

RESTful backend using Django

* Tech Stack

Django

Django REST Framework

OpenWeather API
